# MonjaPro Backend

Backend API com Mercado Pago e Supabase

## Instalação
\`\`\`bash
npm install
\`\`\`

## Configuração
Copie `.env.example` para `.env` e preencha

## Executar
\`\`\`bash
npm run dev
\`\`\`

## Endpoints
- GET /health
- POST /api/mercadopago/create-preference
- POST /api/mercadopago/webhook
